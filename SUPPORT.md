# Support

Thanks for using Project Aria plugins! Here's how to get help.

## Asking questions

For questions about how to use a plugin, integrate it with your workflow, or
about Project Aria more broadly:

- **Project Aria documentation:** <https://facebookresearch.github.io/projectaria_tools/gen2>
- **Community Discord:** <https://discord.gg/projectaria>
- **GitHub Discussions:** open a thread in the [Discussions](../../discussions)
  tab of this repository.

## Reporting bugs

If you've found a bug in a plugin or skill, please open a
[bug report](../../issues/new?template=bug_report.yml). Include reproduction
steps, your environment, and the plugin / skill that's affected.

## Requesting features

Have an idea for a new skill or improvement? Open a
[feature request](../../issues/new?template=feature_request.yml) or a
[new skill proposal](../../issues/new?template=skill_proposal.yml).

## Security issues

**Do not open a public issue for security vulnerabilities.** See
[SECURITY.md](./SECURITY.md) for the responsible disclosure process.

## Commercial support

These plugins are released as community open source. Meta does not offer
commercial support, SLAs, or warranties — see the LICENSE for details.
